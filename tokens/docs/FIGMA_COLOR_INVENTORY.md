### Notable observations

- **Two generations**: "Colors – X" (old) vs "X scale" (new) palettes exist for Red, Blue, and Grey with different hex values for same names
- **Background colors** has 17-step grey (50–950) vs **Grey scale (new)** has 9-step grey — different values at same stops
- **Blue 50 = Blue 300**: `#E4E9F1` appears as Blue 50 in old palette but Blue 300 in new — numbering shift
- **Blue 900 = Blue 800**: `#001533` maps to 900 in old, 800 in new
- **Mislabeled layer**: Grey scale "White" resolves to `#000000` (Black) — Figma layer name bug
- **Functional type** colors match semantic colors by hex but use short names (Red, Yellow, Green, Blue)
